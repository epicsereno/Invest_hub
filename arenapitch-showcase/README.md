# ArenaPitch Showcase

Static product showcase for ArenaPitch, a Shark Tank inspired platform for public pitch discovery and private investor deal rooms.

Open `index.html` directly in a browser. No build step is required.

## Files

- `index.html`: single-page showcase
- `styles.css`: responsive product UI styling
- `app.js`: arena filters and pitch-card rendering
- `docs/`: pitch deck and roadmap notes
